3.2.5. propka.hybrid36¶

3.2.5.1. Hybrid36 PDB-like file format¶

hybrid36 is an alternative PDB format that can encode larger atom numbers. This module provides the decode() functon to parse the atom numbers in hybrid36 “PDB” files.

Functions

Convert an input string of a number in hybrid-36 format to an integer.
